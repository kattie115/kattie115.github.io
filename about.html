Hello there! My name is Katja Šostar, and I’m a UI / UX designer.


I am 24 years old Slovene who currently lives and works in a beatifuly city of Graz, Austria. I enjoy making user friendly interfaces, and solving problems through design - I also dabble in a little code making responsive websites, and sketching icons and illustrations in my off time.

In my free time I enjoy running, gaming, photography, and learning new things.

I come from a traditionaly STEM background, where I studied Infomatics for two years, before realizing that my true passion is design (thanks human-computer class and countless hours of working on project case studies).



Here's what I can do:

UI / UX DESIGN - User flows, user research, wireframing & prototyping, visual design

I love designing simple and consistent web & mobile interfaces focusing on content structure, intuitive UI patterns and simple interactions.


Front End Development - HTML5/CSS3, SASS, JavaScript, JQuery, Bootstrap, Wordpress, KirbyCMS, Jekyll, Git

I like coding my own designs. I impliment my ideas in the browser using HTML, CSS and a touch of JavaScript for interactivity. I love using frameworks like Bootstrap and MDL, but can code my own things from scratch.
